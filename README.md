## üõ† Methodology
We utilize the **Plan-Do-Check-Act (PDCA)** cycle to ensure continuous improvement of the management system:

1. **Plan:** Define scope, objectives, and policies.
2. **Do:** Implement the risk treatment plan and security controls.
3. **Check:** Monitor, measure, and review performance against objectives.
4. **Act:** Take actions to continually improve ISMS performance.
